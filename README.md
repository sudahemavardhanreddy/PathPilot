# PathPilot AI Advisor 🚀

PathPilot is an industrial-grade, AI-powered career and education advisory platform designed to help students discover their aptitude and navigate complex educational pathways.

## Key Features 🌟

-   **AI Aptitude Assessment**: Domain-specific MCQ engine that calculates career alignment across Technical, Commercial, Creative, and Social sectors.
-   **Career Encyclopedia**: A deep-dive database of careers with salary benchmarks, growth projections, and professional toolkits.
-   **Interactive Roadmaps**: 5-stage visualized journeys from Class 10/12 to professional success.
-   **Knowledge & Funding Hub**: Real-time filters for scholarships, competitive exams, and global admission opportunities.
-   **Interview Pilot AI**: Career-specific mock interview simulator with performance analysis.
-   **Career Pivot Simulator**: "What-If" engine to test roadmaps against market disruptions (AI Automation, Recession, etc.).
-   **Global Command Palette**: Lightning-fast fuzzy search (Ctrl+K) for sections, roles, and skills.
-   **Production PWA**: Fully installable as a mobile/desktop app with offline-ready "Stale-While-Revalidate" caching.
-   **Data Freedom**: Built-in JSON backup and profile reset capabilities.

## Tech Stack 🛠️

-   **Frontend**: Vanilla HTML5, CSS3 (Variables, Grid, Flexbox), Javascript (ES6+).
-   **Persistence**: LocalStorage with built-in export/import logic.
-   **PWA**: Service Workers (sw.js) and Web App Manifest.
-   **Performance**: Dynamic asset loading and lazy-rendering for encyclopedia content.

## Getting Started 🏁

1.  Clone the repository.
2.  Open `index.html` in any modern browser.
3.  (Optional) Use a local server (like Live Server) to see PWA features in action.
